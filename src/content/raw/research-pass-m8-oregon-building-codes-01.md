## 2) Claim list with citation anchors

### Confirmed facts (directly supported)

1. PSI’s Oregon CCB exam outline includes **“Oregon Building Codes” with 4 scored items** in the content table. **[T1-1]**
2. The same PSI bulletin states the Oregon test questions cover **laws and business practices**, and the test is **open book** with the NASCLA Oregon guide as the allowed reference. **[T1-1]**
3. ORS Chapter 455 is Oregon’s statewide building code chapter and includes sections covering code purpose/scope, state preemption framework, permit content, and municipal inspection-program administration (e.g., ORS 455.020, 455.040, 455.050, 455.148 as listed in chapter structure and section text). **[T1-2], [T1-3]**
4. Oregon BCD’s adopted-codes page publishes currently effective specialty code editions and interim amendments (including current OSSC and ORSC listings with effective/mandatory dates). **[T1-4]**
5. Oregon BCD jurisdiction program materials state municipalities renew programs in reporting groups under **OAR chapter 918, division 020**, on a four-year cycle. **[T1-5]**
6. CCB contractor guidance includes explicit permit responsibility language that if permits are necessary, contractors/companies must ensure permits are obtained; it also references possible BCD and (in some cases) CCB fines for permit failures. **[T1-6]**

### Inference (explicit synthesis, not a direct quote)

1. For M8 exam prep, the highest-yield tested knowledge is likely **governance and compliance framing** (statewide code authority, local administration, and permit obligations) rather than detailed prescriptive design values from every specialty code chapter. **Inference from [T1-1], [T1-2], [T1-4], [T1-5].**
2. Because code editions and interim amendments change on published cycles, date-stamping code-specific teaching points is essential to avoid drift. **Inference from [T1-4], [T1-5].**
