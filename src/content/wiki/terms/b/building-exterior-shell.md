## Lesson

### Scope focus

Exterior shell work generally covers building envelope components that protect the structure from weather and moisture intrusion. The governing details come from Oregon's adopted specialty code framework. Source: oregon-bcd-adopted-codes-online.

### Authority stack

- **ORS chapter 455** provides statewide building-code administration authority.
- **BCD adopted codes listings** provide currently effective editions and amendment timing.

Use both together when preparing bids and field execution plans. Source: ors-chapter-455, oregon-bcd-adopted-codes-online.

### Practical exam workflow

1. Identify the code family relevant to the scenario.
2. Confirm which edition is currently adopted.
3. Confirm effective/mandatory dates for amendments.
4. Confirm permit expectations with the local jurisdiction.

This workflow prevents outdated-code errors in both exam and real-world settings. Source: oregon-bcd-adopted-codes-online, ors-chapter-455.
