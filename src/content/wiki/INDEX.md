# Oregon CCB Knowledge Wiki — Index

dedicated page in `terms/{letter}/{term}.md`. Terms may appear under multiple
topics when they span domains.

Wiki pages link to each other using double-bracket links (e.g. the slug inside
double square brackets). The `npm run lint:wiki` script validates that all wiki
links resolve to existing term files.
